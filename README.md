# Falcon

Falcon is a personal goal and task tracker web app designed to boost productivity through a modern, calm UI. It helps users manage tasks and goals, visualize progress, and stay organized without authentication.

## Objectives
- Track tasks and personal goals effectively.
- Provide a responsive experience across mobile, tablet, and desktop.
- Offer a calendar view for date-based planning.
- Support custom themes and categories for goals and tasks.
- Maintain clarity, calm, and efficiency in design.

## Core Features
### Task Management
- Add tasks with description, due date, and priority (low/medium/high).
- Edit and delete tasks.
- View tasks in list or calendar formats.
- Search tasks by keyword.

### Goal Setting
- Define yearly, monthly, and daily goals.
- Categorize tasks and goals with themes (e.g., Spiritual, Physical Fitness, Academics, Career).
- Create and manage custom categories.

### Progress Tracking
- Dashboard with progress metrics for today, this week, this month, and this year.
- Simple visual charts for progress indicators.

## UI/UX
- Modern, minimalistic design with a default dark theme and soft accents.
- Calendar-first layout with smooth day expand/zoom interactions.
- Clean task cards with subtle animations.

## Accessibility
- Fully responsive layout across devices.
- No authentication required (personal use only).
