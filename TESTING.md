# Testing

## How to Run
From `falcon/codebase`:

```bash
npm test
npm run build
```

## PRD Must-Have Coverage
- Task management (add tasks, search/filter): `src/tests/tasks.test.tsx`
- Goal setting (add goals, levels): `src/tests/goals.test.tsx`
- Calendar view & day zoom: `src/tests/calendar.test.tsx`
- Custom categories and themes: `src/tests/settings.test.tsx`
- Progress tracking dashboard metrics: `src/tests/dashboard.test.tsx`
